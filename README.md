# STUDENT MANAGEMENT SYSTEM

 Overview
 
 The Student Management System (SMS) is a simple yet practical Python-based console application designed to manage student records efficiently. It helps maintain student information such as Roll Number, Name, Age, and Marks.
The system supports adding, viewing, searching, updating, and deleting student records, while storing data permanently in a CSV file.

This project is ideal for beginners, college submissions, and real-world learning of file handling, modular programming, OOP concepts, and CRUD operations in Python.

# FEATURES :

1. Add new student details
 
2. Display all saved records
 
3. Search for a student by roll number
 
4. Update existing student information
 
5. Delete student records
 
6. Permanent storage using CSV file
 
7. Fully modular code (5 Python modules)
 
8. Runs easily in VS Code or any Python IDE
   
 

# TECHNOLOGIES AND TOOLS USED :
